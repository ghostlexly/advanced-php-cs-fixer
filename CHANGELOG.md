# Change Log

All notable changes to the "advanced-php-cs-fixer" extension will be documented in this file.

## [1.0.0] - 2025-11-17

### Added

- Initial release
- Real-time diagnostics for php-cs-fixer errors and warnings
- Document formatting support using php-cs-fixer
- Configuration options for executable path, config file, and rules
- Commands to manually fix or check current PHP file
- Support for custom php-cs-fixer configuration files
- Option to enable/disable diagnostics and formatting independently
- Support for risky rules
